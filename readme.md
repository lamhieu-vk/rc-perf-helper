tools to optimize performance when using React


## install

```bash
# use npm
$ npm install rc-perf-helper

# or yarn
$ yarn add rc-perf-helper
```

## usage

### dynamicFn

```js
import { dynamicFn } from 'rc-perf-helper'
```

cache your React event listeners to improve performance
